# 🧟 Zombie Escape

Zombie Escape is a fast-paced 2D survival game built with HTML, CSS, and JavaScript. Survive endless waves of zombies, collect coins, manage your stamina, and see how long you can stay alive.

---

## Features

- Endless zombie waves
- Normal, Fast, Tank, and Boss zombies
- WASD and Arrow Key movement
- Sprint system with stamina
- Dash ability
- Mouse aiming and shooting
- Health system
- Coin collection
- Health pickups
- Blood particle effects
- Animated crosshair
- Wave progression
- Pause menu
- Game over screen
- Survival timer
- Responsive design

---

## Controls

| Action | Key |
|---------|-----|
| Move | WASD / Arrow Keys |
| Sprint | Shift |
| Dash | Space |
| Shoot | Left Mouse Button |
| Pause | P |

---

## How to Play

1. Open `dodge.html` in your browser.
2. Click **Start Game**.
3. Survive as long as possible.
4. Shoot zombies before they reach you.
5. Collect coins and health pickups.
6. Every few waves the game becomes harder.
7. Boss zombies appear every 5 waves.

---

## Game Mechanics

### Zombies

- 🟥 Normal Zombie
- 🟧 Fast Zombie
- 🟫 Tank Zombie
- 👹 Boss Zombie

Each zombie type has different health, speed, and rewards.

### Coins

Coins randomly drop from defeated zombies and increase your total score.

### Stamina

Sprinting consumes stamina.

When stamina reaches zero you must wait for it to regenerate before sprinting again.

### Health

The player starts with 5 health.

Health packs occasionally spawn around the map.
